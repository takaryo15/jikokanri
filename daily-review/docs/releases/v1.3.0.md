# daily-review v1.3.0

v1.3は、毎日の入口を`home`へまとめ、ChatGPTとの安全なJSON往復、タスク繰越、
検証可能なbackup/restore、週次・月次分析、schedulerを一つのローカル運用へ
統合するリリースです。

## 主な変更

- preview、confirmation、idempotency、監査ログ付きCommand API
- Main最大3件と最低限を守るタスク・指示書操作
- SHA-256 manifest付きbackupと、preview-first restore
- 複製しないrollover
- 火曜〜月曜の週次、前月比較付き月次、明示承認とstale検出
- missed run、retry、quiet hours、重複slot防止を持つscheduler
- macOS launchdの明示install
- setup、onboarding、v1.0〜v1.3 migration、横断doctor

## 最短の使い方

```bash
daily-review setup
daily-review migrate check
daily-review doctor
daily-review home
daily-review close-day --clipboard --dry-run
```

## 移行

本番データを直接試さず、backupを検証し、コピー上でmigrationとrestoreを
リハーサルしてください。詳細は
`docs/migration-v1.0-to-v1.3.md`を参照してください。

## 互換性

既存CLIとv1.0〜v1.2のJSONを維持します。破壊的なschema一括変換はありません。
廃止コマンドはありません。旧来の個別コマンドは高度な操作用として残ります。

## 制約

- 外部通知senderは未同梱（console/fileのみ）
- Web UI、クラウド同期、複数端末同期なし
- 自然言語整理は保守的なルールベース
- launchd統合はmacOSのみ
- schedulerのjob timeoutはprocess強制終了ではなく、stale lock検出と次回pollで回復
- missed runは`run_once`、`skip`、`notify_only`に対応し、全取りこぼし枠を再生する`run_all`は対象外

## ロールバック

migration前backupを`backup verify`し、`restore preview`で差分を確認してから
期限付きtokenでapplyします。復元前にも自動backupが作成されます。
